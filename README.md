# PSPDFKit for iOS - SwiftUI Document Browser

This is the SwiftUI document browser example app for [PSPDFKit for iOS](https://pspdfkit.com/pdf-sdk/ios/). This project shows how to display a document browser in SwiftUI, allowing you to open and create PDF documents.

## Requirements

* The latest [Xcode](https://developer.apple.com/xcode/).

## Getting Started

* Click on the green "Code" button and select the "Open with Xcode" option.<sup>[1](#footnote1)</sup>
* Follow the steps in Xcode to clone and open the project.
* Build and run.
